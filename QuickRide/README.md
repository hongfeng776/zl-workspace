# 捷行出行 - QuickRide

一个现代化的出行服务平台，采用前后端分离架构。

## 技术栈

### 后端
- Node.js + Express
- TypeScript
- SQLite (数据库)
- JWT (身份认证)
- Multer (文件上传)

### 前端
- Vue 3 + TypeScript
- Vite
- Vue Router
- Pinia (状态管理)
- Element Plus (UI组件库)
- Axios (HTTP请求)

## 功能特性

- ✅ 手机号 + 短信验证码快捷注册
- ✅ 验证码快捷登录
- ✅ 密码找回功能
- ✅ 账号安全退出
- ✅ 身份实名认证
- ✅ 账号申诉
- ✅ 历史行程关联

## 快速开始

### 启动后端服务

```bash
cd backend
npm install
npm run dev
```

后端服务运行在 http://localhost:3000

### 启动前端应用

```bash
cd frontend
npm install
npm run dev
```

前端应用运行在 http://localhost:5173
